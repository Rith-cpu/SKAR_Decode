FTC Team SKAR 26249 - DECODE Season

Welcome to the official code repository for FTC Team 26249 for the DECODE (2025–2026) season. This repository contains all the programming resources used to control our robot during autonomous and teleoperated periods.

Contents

TeamCode – Main robot code for autonomous and teleop modes.

Libraries – Any external libraries or helper classes used in the project.

README.md – Documentation for setting up and understanding the code.

Features

Mecanum Drive Control – Omnidirectional movement using joystick input.

Autonomous Routines – Preprogrammed strategies for scoring points during autonomous.

TeleOp Control – Manual control for driving, intaking, and lifting mechanisms.

Claw and Arm Mechanisms – Full control of manipulator components.

Proportional Motor Control – Smooth speed adjustments based on joystick input.

Getting Started
Requirements

Android Studio – Recommended IDE for FTC development.

FTC SDK 8.x – Make sure to use the version compatible with the DECODE season.

Java 11 – Required for compiling and running the robot code.

Installation

Clone this repository:

git clone https://github.com/[YourUsername]/FTC-Decode-Code.git


Open the project in Android Studio.

Connect your robot controller phone or simulator.

Build and deploy to run the code.
